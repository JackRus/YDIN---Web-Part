YDIN

Attendance Management System. Does It for You.

Features

    QR Code Scanner (iOS App):
  • Scanns QR Codes (QRC)
  • Time control (checks when QRC was generated and when it was scanned)
  • QRC Origin Check (accepts only YDIN-generated QRCs)
  • Event Time control (lets to scan QRC only at a certain time frames, when Events occure)
    • WEB Interface
  • Individual attendance rate for every event
  • Separate attendance by event type
  • Compact vertical table header
  • Separate attendance for tracks
  • Custom event names
  • Statistics: deatiled statistics page for each event type
    • QRC Generator
  • QRC is dynamicaly generated
    • Records Management
  • Add Event
  • Add Attendance Record
  • Delete Event
  • Delete Attendance Record
  • Add Student
  • Delete Student
    • Database
  • Prevents from creating dublicate attendance records
  • Records automatically created with php-script after QRC was scanned
